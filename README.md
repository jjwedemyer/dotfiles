# dotfiles

### Installation
```sh
git clone repo ~/.dotfiles && cd ~/.dotfiles
./install.sh
```

### dependencies
- zsh, git
- zsh antigen
- [hammerspoon](https://www.hammerspoon.org/) on macOS (needs to be installed manually for now)
